# Pfariso Netshambidi — Portfolio

A single-page portfolio site (`index.html`) — About, Skills, Projects, Contact.

## Add your photo

Drop your headshot into the `assets` folder and name it `profile.jpg`. It'll automatically appear in place of the initials avatar. (If you use a different filename/format, update `assets/profile.jpg` on the `<img id="profileImg">` line in `index.html`.)

## Host it free on GitHub Pages

1. Go to [github.com](https://github.com) and log in (or create a free account).
2. Click the **+** icon top-right → **New repository**.
3. Name it exactly `yourusername.github.io` (replace `yourusername` with your actual GitHub username) — this gives you the cleanest URL. Set it to **Public**. Click **Create repository**.
4. On the new repo page, click **uploading an existing file**.
5. Drag in `index.html`, the `assets` folder (with your photo inside), and drop the `PUT_YOUR_PHOTO_HERE.txt` file (or just don't upload it once your photo is in place).
6. Click **Commit changes**.
7. Go to **Settings** → **Pages** (left sidebar). Under "Build and deployment," source should already be set to **Deploy from a branch**, branch `main`, folder `/ (root)`. Save if needed.
8. Wait 1–2 minutes, then visit `https://yourusername.github.io` — your site is live.

Any time you want to update the site, just edit `index.html` and re-upload it to the same repo (or use "Edit" directly on GitHub).
